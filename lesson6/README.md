# Custom Dropdown Menu

## Функціонал
- **Single select** — вибір однієї технології з групованого списку
- **Multi select** — вибір декількох з чекбоксами, Select All / Clear All, теги обраних
- Пошук у реальному часі з підсвіткою результатів
- Клавіатурна навігація: `↑↓` — переміщення, `Enter` — вибір, `Esc` — закриття
- Закриття при кліку поза dropdown
- Badge counter з кількістю обраних

## Дані
4 категорії по 10 технологій: Фронтенд, Бекенд, Databases, DevOps

## Запуск
Відкрити `index.html` через Live Server у VS Code
